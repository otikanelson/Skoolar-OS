## Screenshot Tips

1. **Use Full Browser Window** - Maximize browser for best presentation
2. **Consistent Size** - Take all screenshots at same resolution
3. **Clean Browser** - Hide bookmarks bar, extensions for clean look
4. **Order Matters** - Follow the flow: Landing → Login → Admin → Teacher → Parent
5. **Highlight Key Features** - Focus on the institutional, structured design
