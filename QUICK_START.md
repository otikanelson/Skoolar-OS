## Get Your Screenshots in 5 Minutes!

### Step 1: Start the Server (30 seconds)

```bash
cd educore-os/frontend
npm run dev
```

Wait for: `Local: http://localhost:5173/`

---

### Step 2: Open Your Browser

Visit: **http://localhost:5173**

---
